3Dゲーム

ブラウザで動作するゲームやインタラクティブ作品をまとめています。2D と 3D の表現、 ルール実装、UI 設計、アクション要素の違いを見比べられます。

Overview

遊べる作品をまとめて比較できるページです

パズル、ボードゲーム、アクションまで、ブラウザでそのまま触れる作品を公開しています。 UI の作り方やルール処理、3D 表現、入力の扱いなど、作品ごとの違いも合わせて確認できます。

  • 2D / 3D の幅を持たせたゲームを公開
  • CPU 対戦やロジック重視の作品も収録
  • ブラウザだけでそのまま遊べる構成
  • UI とルール処理の違いを見比べやすい一覧

軍議 CPU 対戦、手駒配置、ツケの 3D 表現まで試せるブラウザ版です。 2048 定番パズルの基本ルールを短時間で試せます。 Breakout アクション性のある作品として操作感を確認できます。 SameGame 3D 立体パズルならではの視点操作と消去ルールを試せます。
