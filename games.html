ゲーム

ブラウザで触れるゲーム作品の入口です。

パズル、対戦、アクションをすぐ試せて、UI やルール設計の違いも見比べられます。

  • 2D / 3D の幅を持たせたゲームを公開
  • CPU 対戦やロジック重視の作品も収録
  • ブラウザだけでそのまま遊べる構成
  • UI とルール処理の違いを見比べやすい一覧

まず触ってほしい 4 作品

軍儀 CPU 対戦、手駒配置、ツケの 3D 表現まで試せるブラウザ版です。 2048 定番パズルの基本ルールを短時間で試せます。 Breakout アクション性のある作品として操作感を確認できます。 SameGame 3D 立体パズルならではの視点操作と消去ルールを試せます。

新規公開の軍儀は、9x9 盤面、重ね置きの 3D 表現、CPU 対戦までを含む作品です。
