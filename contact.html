興味があるテーマ

いま特に関心が強いのは、仕組みが見える題材をインタラクティブに伝えることです。 ラボ、ゲーム、バイク図鑑で扱っているテーマがそのまま現在の興味領域になっています。

Vehicle / Moto

車両とバイクの構造・挙動

車両物理、エンジン、駆動系、バイク図鑑のように、仕組みや違いが見える題材に強く惹かれています。

Mathematics

数学を図と動きで伝えること

数、変化、構造のような抽象概念を、図やインタラクションに置き換えて理解しやすくするテーマに興味があります。

Fluid / Graphics

流体とリアルタイム表現

PB-MPM、水面、煙、インクのように、挙動の面白さと見た目の説得力を両立する表現に取り組んでいます。

Interactive Web

触って理解できる Web 体験

読むだけでなく、操作しながら内容を理解できるブラウザ作品や学習コンテンツを作ることに関心があります。

得意なテーマ

現在の成果物群の中でも、実際に手を動かして積み上げてきた比重が大きい領域です。 AI と協業しながら、現実の現象や構造を言語化し、それを Web ページの仕組みとインタラクションに落とし込む進め方を強みとしています。

Browser Games

ブラウザゲームの実装

2048、Breakout、SameGame 3D、軍儀のように、ルール設計から実装まで一貫して形にすることを得意としています。

3D / Simulation

3D とシミュレーションの見せ方

エンジン 3D や流体デモのように、複雑な動きや状態をブラウザ上で見せる構成や調整に強みがあります。

AI Collaboration

現象の言語化とインタラクション設計

現実の挙動や仕組みを言葉で整理し、AI と協業しながら Web ページ上の動きや操作体験へ変換することを得意としています。車両物理、数学可視化、流体デモでもこの進め方を軸にしています。

Catalog / Comparison

比較しやすい情報のまとめ方

バイク図鑑のように、一覧、比較、詳細の流れで複数の対象を見やすく整理する情報設計にも取り組んでいます。
