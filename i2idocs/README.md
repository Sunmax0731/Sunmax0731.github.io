# i2idocs

I2I工程支援Webアプリの構想資料を用途別に整理したディレクトリです。

## 構成
- `guides/`: 実装ルールとデザイン方針
- `specs/`: 要件、画面設計、ワイヤーフレーム、プロンプト設計
- `schemas/`: アプリで扱う主要エンティティの JSON Schema

## 主な参照順
1. `guides/AGENTS.md`
2. `guides/Design.md`
3. `specs/requirements.md`
4. `specs/screen-design.md`
5. `specs/prompt-sheet-workflow.md`

## 備考
- `specs/prompt-sheet-workflow.md` が詳細版、`specs/prompt-sheet-workflow-summary.md` が要約版です。
- `schemas/schema-index.json` はスキーマ一覧の正本です。
